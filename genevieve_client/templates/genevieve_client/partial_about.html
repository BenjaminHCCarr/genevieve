Who runs Genevieve?

Genevieve is written and managed by the same nonprofit organization that created and runs Open Humans.

Genevieve is open source.

Genevieve's source code is shared on GitHub with an MIT (Expat) license.

In addition, notes contributed via Genevieve are public domain (CC0). Notes are stored in the GenNotes database as public domain material (CC0) and are available via the GenNotes API. (The GenNotes server is also operated by our organization, and is also open source.)
